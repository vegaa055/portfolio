Game Overview

This project came about through an independent study while working towards my bachelors. I became curious about Python being used in game development and how I could go about doing so. I began to research and discovered PyGame, a cross-platform set of Python modules intended to be used in game development. Perfect! I started reading the documentation and set up the development environment.

Technologies Used

  • Python
  • PyGame

Gameplay Features

  • Increasing number of enemies
  • Power-ups to enhance player abilities
  • High score tracking
