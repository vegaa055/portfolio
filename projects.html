Cryptid Hunter: Rogue - Unity/C#

2D roguelike dungeon crawling shooter. Featuring enemies with A* pathfinding/AI, procedurally generated dungeons, and visual effects using shaders. In addition, I added 2D lighting and shadows to give a little more depth. Built with Unity & C# and taking advantage of Unity's ScriptableObjects.

Space Force - Python/PyGame

2D retro-inpsired space-shooter game where you face an increasing number of enemies until you are destroyed. Developed in Python using the PyGame library. This game was a challenge to develop without an engine but it was also a learning experience. Being that Python is rarely used in game development, I was curious to see what can be done. I was actually quite pleased with the end product!
